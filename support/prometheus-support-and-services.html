Covered Components

DIXPAI Support includes assistance with:

  • Prometheus core functionality
  • Configuration and deployment
  • Data scraping and collection
  • Alerting rules and configuration
  • Querying and data analysis (PromQL)
  • Performance tuning and optimization
  • Scaling and high availability setups
  • Integration with other monitoring tools

Additional supported components:

  • Alertmanager
  • Grafana integration and dashboard setup
